## 📌 Project Description
This is a desktop-based application developed using Java Swing and MySQL.  
The system helps in managing and checking the availability of medicines in a pharmacy.

---

## 🚀 Features
- User Registration and Login System
- Add Medicine Details
- Update Medicine Information
- Delete Medicine Records
- View Available Medicines

---

## 🛠️ Technologies Used
- Frontend: Java Swing
- Backend: MySQL
- Connectivity: JDBC

---

## 🎯 Purpose
This project reduces manual work and helps in efficient management of medical inventory.
